
# Código fuente

Esta carpeta contiene el archivo fuente principal del proyecto en formato **R Markdown**.

## Contenido

- `TFG (para html).Rmd`: documento fuente editable del análisis.

## Descripción

Este archivo integra texto, código y resultados en un único flujo de trabajo reproducible. Desde él se estructura el análisis del TFG y se generan salidas en otros formatos, como HTML.

## Objetivo

Centralizar la parte editable y reproducible del proyecto, permitiendo:

- modificar el análisis,
- actualizar resultados,
- regenerar informes,
- y mantener trazabilidad entre código, texto y visualizaciones.

## Nota

Cualquier cambio sustancial en el contenido analítico o narrativo del trabajo debería realizarse a partir de este archivo.
