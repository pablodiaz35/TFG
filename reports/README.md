
# Informes generados

Esta carpeta contiene las salidas generadas a partir del archivo fuente del proyecto.

## Contenido

- `TFG--para-html-.html`: versión en HTML del trabajo.

## Descripción

El archivo HTML permite visualizar el contenido del análisis en navegador, incluyendo texto, resultados y gráficos generados desde el documento fuente en R Markdown.

## Objetivo

Facilitar una revisión rápida y accesible del trabajo sin necesidad de abrir el archivo fuente ni recompilar el documento.

## Nota

Los archivos de esta carpeta son resultados derivados del contenido de `src/` y no constituyen el punto principal de edición del proyecto.
