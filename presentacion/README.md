
# Presentación

Esta carpeta contiene la presentación del Trabajo de Fin de Grado.

## Contenido

- `PabloDiaz_TFG.pptx`: presentación utilizada para resumir y exponer el estudio.

## Descripción

La presentación sintetiza los principales elementos del trabajo:

- la pregunta de investigación,
- el contexto normativo de la ley SB-826,
- la muestra, periodos y herramientas empleadas,
- los principales resultados sobre desigualdad salarial,
- y las conclusiones finales.

## Objetivo

Servir como material de apoyo para la defensa, exposición o comunicación resumida del TFG.
